# Piotr Ponikiewski Portfolio

One-page portfolio built with Astro, React, TypeScript and Tailwind CSS.

## Scripts

```bash
pnpm install
pnpm dev
pnpm build
```

## Structure

- `/pl` - Polish version
- `/en` - English version
- `src/i18n/content.ts` - page copy and project data
- `public/cv/` - downloadable CV
- `public/images/` - profile image and future project screenshots

Project screenshots are currently represented by placeholders and can be replaced later.
